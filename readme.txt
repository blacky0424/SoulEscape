魂が地獄から脱出し、天国を目指して進むシューティングゲーム
ステージが１〜３まであり、ステージ３を突破すればゲームクリア
クリアまでの距離が設定されていて、距離が０に近づくとボスが出現（距離は自動的に縮まっていく）
様々な動きをして流れてくる弾を避けるか破壊しながら進み、ボスを倒すことでステージクリア

遊び方:
←,Aキー:左移動　→,Dキー:右移動 ↑,Wキー:上移動　↓,Sキー:下移動　SPACEキー,Pキー:弾発射
弾は左上の弾ゲージが満タンの時のみ発射できる。発射するとゲージが１つ減る（自動回復）。
弾ゲージの上の数字が残り体力。
右下に現在のスコア、右上に目的地（ステージボス）までの残り距離が表示されている。

こだわった点
処理の重いオブジェクトの生成や破棄を多用せず、オブジェクトプールを意識した。
